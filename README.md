# Weather-dashboard
# 🌦️ Weather Forecast App

A simple Python-based weather application that fetches and displays current weather conditions and a 5-day forecast using weather API data.

## Features

-  Current temperature
-  Feels-like temperature
-  Weather condition
-  Humidity
-  Wind speed and direction
-  Atmospheric pressure
-  Visibility
-  5-day weather forecast
-  Postal/PIN code based search
-  Clean terminal interface with emojis

## Technologies Used

- Python 3
- Requests Library
- Weather API
- Json File Management

## Installation

1. Clone the repository

```bash
git clone https://github.com/Ruchit-warade/weather-dashboard.git
cd weather-app
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Find All Places in code written with <your_api_key>
4. Create your free api key by signing in on openweatherapp website
5. Add your API key in the source code.

6. Run the application

```bash
python weather.py
```
